## radar

### Key Bindings

### Any Tab
|  Key  |  Action                    |
| ----- | -------------------------- |
| F1    | Move to Radar screen       |
| F2    | Move to Coverage screen    |
| F2    | Move to Airplanes screen   |


### Map or Coverage
|  Key  |  Action                    |
| ----- | -------------------------- |
| -     | Zoom out                   |
| +     | Zoom in                    |
| Up    | Move Map Up                |
| Down  | Move Map Down              |
| Left  | Move Map Left              |
| Right | Move Map Right             |
| Enter | Reset Map                  |

### Airplanes
|  Key  |  Action                    |
| ----- | -------------------------- |
| Up    | Move selection upward      |
| Down  | Move selection downward    |
| Enter | Center Map tab on aircraft |
