### Key Bindings

|  Key  |  Action                   |
| ----- | ------------------------- |
| F1    | Move to Radar screen      |
| F2    | Move to Coverage screen   |
| -     | Zoom out                  |
| +     | Zoom in                   |
| Up    | Move Map Up               |
| Down  | Move Map Down             |
| Left  | Move Map Left             |
| Right | Move Map Right            |
| Enter | Reset Lat/Long(Map)       |
